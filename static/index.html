AI 회의 어시스턴트 시작하기
1 목소리 등록 (선택)
왼쪽 패널 직접 등록 탭에서 이름을 입력하고 등록 버튼을 누른 뒤 5초간 말하세요. 참여자 전원 등록 후 회의를 시작하면 발화자를 자동으로 구분합니다.
• 저장된 프로파일이 있다면 자동 식별 탭을 사용하세요.
2 회의 시작
언어, 템플릿, 정확도를 선택하고 • 회의 시작 버튼을 누르세요.
• 마이크가 자동으로 켜지고 실시간 전사가 시작됩니다.
• ⏸ 일시정지 — 잠깐 멈출 때
• ✕ 취소 — 기록 없이 종료
• ■ 종료 + AI 요약 — 정식 종료 및 분석
3 Claude에게 질문
회의 중 왼쪽 하단 텍스트창에 질문을 입력하고 전송(⌘↵)하세요.
예) "지금까지 결정된 것 요약해줘"
예) "Asana에서 이재헌 담당 태스크 찾아줘"
예) "다음 회의 어젠다 초안 만들어줘"
💡 자동으로 동작하는 것들
✏️ STT 교정 — 음성 인식 오류 자동 수정
📍 주제 감지 — 90초마다 대화 주제 변경 감지
🤖 자동 개입 — "마감이 언제야?", "담당자가 누구?" 패턴 자동 답변
📊 품질 피드백 — 10분마다 회의 진행 상태 분석
