AI 설정

Claude Code CLI + MCP 연결

AI 분석·자동 개입·Asana 연동은 Claude Code CLI 설치와 MCP 설정이 필요합니다.

🤖
AI 기능은 Claude CLI 없이 동작하지 않습니다

이 시스템의 핵심 가치인 STT 교정, 자동 개입, 회의 요약, 음성 트리거는 모두 Claude Code CLI를 subprocess로 호출해서 동작합니다. Claude CLI가 설치·인증되어 있지 않으면 전사는 되지만 AI 분석 기능 전체가 비활성화됩니다. Claude Pro, Team, 또는 API 플랜이 필요합니다.

❌ CLI 미설치 시
STT 교정 없음 · 요약 없음 · 자동 개입 없음
✅ CLI 설치 완료 시
모든 AI 기능 + MCP 연동 + 음성 트리거 사용 가능
1

Node.js 설치 확인

Claude Code CLI는 npm으로 설치합니다. Node.js 18 이상이 필요합니다.

node --version # v18 이상 필요 # 없으면: https://nodejs.org 에서 설치
2

Claude Code CLI 설치

npm install -g @anthropic-ai/claude-code
3

Anthropic 계정 인증

처음 실행 시 브라우저가 열리면서 Anthropic 계정으로 로그인합니다. Claude Pro / Team / API 플랜이 필요합니다.

claude # 최초 실행 → 브라우저 인증
4

설치 확인

claude --version # 버전 출력되면 완료 claude -p "안녕" # 응답 오면 정상 작동

⚠️ UI 상단 🔧 버튼 → 시스템 상태에서 Claude CLI 연결 여부를 확인할 수 있습니다.

MCP(Model Context Protocol)는 Claude가 외부 서비스를 직접 호출할 수 있게 해주는 플러그인입니다. 회의 중 "Asana에서 이재헌 담당 태스크 찾아줘" 같은 요청을 처리합니다.

1

MCP 설정 파일 위치 확인

Claude Code의 MCP 서버는 ~/.claude.json 또는 프로젝트의 .claude/settings.json에 정의합니다.

# 설정 파일 열기 cat ~/.claude.json # 전역 설정
2

Asana MCP 연결 (태스크 생성·조회)

# ~/.claude.json 에 추가 { "mcpServers": { "asana": { "command": "npx", "args": ["-y", "@modelcontextprotocol/server-asana"], "env": { "ASANA_ACCESS_TOKEN": "your_token_here" } } } }

토큰 발급: app.asana.com/0/my-apps → Personal Access Token

3

Slack MCP 연결 (채널 메시지 조회·전송)

"slack": { "command": "npx", "args": ["-y", "@modelcontextprotocol/server-slack"], "env": { "SLACK_BOT_TOKEN": "xoxb-...", "SLACK_TEAM_ID": "T..." } }
4

Google Calendar MCP (일정 조회·생성)

"google-calendar": { "command": "npx", "args": ["-y", "@anthropic-ai/mcp-server-google-calendar"] # 첫 실행 시 OAuth 브라우저 인증 }
5

MCP 연결 확인

npm install -g @anthropic-ai/claude-code claude # 최초 인증

마이크 접근 권한 허용

5

start.command 더블클릭

브라우저가 자동으로 열리며 http://localhost:5555로 접속됩니다.

🔒 HTTPS로 접속하려면

모바일 접속 또는 보안 연결이 필요한 경우 아래 두 가지 방법 중 선택하세요.

방법 1 — 터미널에서 직접
HTTPS=1 python3 server.py
방법 2 — start.command 마지막 줄 변경 (항상 HTTPS)
# python3 server.py ← 이 줄을 아래로 교체 HTTPS=1 python3 server.py

⚠️ 최초 실행 시 인증서 자동 생성 → 브라우저 경고에서 고급 → 계속 진행 클릭

1

Python 3.10+ 설치

python.org에서 Windows 설치 파일을 다운로드합니다. 설치 시 "Add Python to PATH" 체크박스를 반드시 활성화하세요.

2

자동으로 브라우저가 열리며 http://localhost:5555로 접속됩니다.

🔒 HTTPS로 접속하려면

set HTTPS=1 && python server.py

rem python server.py ← 이 줄을 아래로 교체 set HTTPS=1 && python server.py

⚠️ 최초 실행 시 인증서 자동 생성 → 브라우저 경고에서 고급 → 계속 진행 클릭

단계별 사용 방법

처음 사용하더라도 아래 순서대로 따라하면 바로 시작할 수 있습니다.

1

서버 실행

start.command (Mac) 또는 start_windows.bat (Windows)를 더블클릭합니다. 브라우저가 자동으로 열리며 http://localhost:5555로 접속됩니다.

처음 실행 시 Whisper 모델 로드 약 30초 소요
2

목소리 등록 첫 사용 시 강력 권장

목소리를 미리 등록하면 회의록에 "이재헌: ..." 형태로 발화자가 정확히 표시됩니다. 등록 없이 시작하면 "화자1", "화자2"처럼 익명으로 기록되어 나중에 누가 한 말인지 구분하기 어렵습니다.

🎤 직접 등록 (권장)
지금 이 자리에서 5초간 목소리 녹음
회의 참석자 전원 등록 후 시작
가장 정확한 화자 구분
🔄 자동 식별
이전에 등록한 프로파일로 자동 매칭
한 번만 등록하면 이후 회의는 자동
신규 참석자는 미매칭으로 처리
🎤 직접 등록 단계별 방법
1
왼쪽 패널에서 직접 등록 탭을 선택합니다.
2
이름 입력창에 첫 번째 참석자 이름을 입력합니다. (예: 이재헌)
3
등록 버튼을 클릭하고 상태바에 🔴 녹음 중... 표시가 나타나면 해당 인물이 5초간 자연스럽게 말합니다.
예: "안녕하세요, 저는 이재헌입니다. 오늘 회의 잘 부탁드립니다."
4
등록 완료 메시지를 확인한 뒤 다음 참석자도 동일하게 반복합니다.
5
참석자 전원 등록 후 ● 회의 시작 버튼을 누릅니다.
💡 잘 되는 팁
· 조용한 환경에서 등록할수록 정확도가 높아집니다
· 한 사람씩 번갈아 가며 등록하세요 (동시 발화 금지)
· 등록한 목소리 프로파일은 voices/ 폴더에 저장되어 다음 회의에서도 자동 사용됩니다

자주 묻는 질문

도입 전에 가장 많이 받는 질문들을 모았습니다.

음성과 원본 텍스트는 로컬에만 저장됩니다. 마이크 → Whisper 전사 → 회의록 저장까지 모두 내 컴퓨터 안에서 처리됩니다.

단, STT 교정·요약·자동 개입 기능 사용 시 발화 텍스트 일부가 Claude Code CLI를 통해 Anthropic 서버로 전송됩니다. 음성 파일 자체는 절대 외부로 나가지 않습니다.

AI 기능 사용 시 필요합니다. Claude Code CLI는 claude.ai 계정으로 OAuth 인증하며 아래 플랜이 지원됩니다.

✅ Claude Pro ($20/월) ✅ Claude Team ✅ Claude Max ✅ Anthropic API

Claude Code CLI는 API 키 방식이 아닌 claude.ai 계정 인증 방식입니다. 첫 실행 시 브라우저가 열리면서 로그인하면 자동 설정됩니다.

🛠️ 문제 해결

브라우저 HTTPS 경고

"연결이 안전하지 않습니다" 화면에서 → 고급 → localhost(안전하지 않음)로 계속 이동 클릭 (자체 서명 인증서이므로 정상)

HTTPS 인증서 재생성

rm cert.pem key.pem # 서버 재시작 시 자동 재생성

cryptography 미설치 시 HTTP로 실행됨

pip3 install cryptography # macOS/Linux pip install cryptography # Windows

마이크 권한 오류 (macOS)

시스템 설정 → 개인정보 보호 → 마이크 → 터미널 체크

마이크 권한 오류 (Windows)

설정 → 개인 정보 → 마이크 → 앱 마이크 액세스 허용

모바일에서 마이크가 안 될 때

HTTPS가 필요합니다. 1. cryptography 설치 후 서버 재시작 2. 브라우저에서 https://[PC IP]:5555 접속 3. 인증서 경고 → 계속 진행

모델 다운로드 실패

python3 -c "from faster_whisper import WhisperModel; WhisperModel('large-v3', device='cpu', compute_type='int8')"

발화자 구분 안 됨 (resemblyzer 없음)

pip3 install resemblyzer # macOS/Linux pip install resemblyzer # Windows

발화 감지 정확도 낮음 (webrtcvad 없음)

pip3 install webrtcvad # macOS/Linux pip install webrtcvad # Windows

소음 제거 안 됨 (noisereduce 없음)

pip3 install noisereduce # macOS/Linux pip install noisereduce # Windows

포트 충돌

# macOS/Linux lsof -i :5555 kill -9 [PID] # Windows netstat -ano | findstr :5555 taskkill /PID [PID] /F

Claude CLI 미설치

npm install -g @anthropic-ai/claude-code
